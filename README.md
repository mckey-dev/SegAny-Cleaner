# SAM2 Image Cropping Tool (Ver 1.1.0)

Metaの「Segment Anything Model 2 (SAM2)」を活用した画像切り抜きツールです。
全自動解析と「消しゴム編集」に加え、プロ仕様の境界ぼかし機能を搭載しています。

## 主な更新点 (Ver 1.1.0)
- **境界ぼかし機能**: 切り抜き実行時にエッジをピクセル単位で滑らかに加工可能。
- **自動PNG変換**: 入力がJPGでも、透過情報を保持するために強制的にPNG形式で書き出し。
- **堅牢性の向上**: 極端な解析しきい値設定によるシステムクラッシュを防止するガード処理を実装。
- **UIの整理**: 調整項目をアコーディオンに集約し、作業スペースを最適化。

## 構成
- **ノートブック**: `SegAny-Cleaner.ipynb`
- **主要スクリプト**: `sac.py`
- **モデル保存先**: `/tmp/sac/models/`
- **一時作業領域**: `/tmp/sac/`

## 実行方法
1. Paperspaceのリポジトリ内で **`SegAny-Cleaner.ipynb`** を開きます。
2. 全てのセルを順に実行してください。
3. 最後に表示される `public URL` または `local URL` をクリックしてUIを起動します。

## 使い方
1. **解析**: 画像をアップロードし「解析実行」をクリック。
2. **選択**: 解析画像の上を適当になぞり「マスク生成」をクリック。
3. **修正**: マスク結果の不要な部分をペン（消しゴム）でなぞって消去。
4. **調整**: 「切り抜き調整」でぼかし（px）や反転を設定。
5. **出力**: 「切り抜き実行」で透過PNGをダウンロード。
